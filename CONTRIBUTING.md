# Contributing to bugstack-ruby

Thank you for your interest in contributing!

## Development Setup

```bash
git clone https://github.com/MasonBachmann7/bugstack-ruby.git
cd bugstack-ruby
bundle install
```

## Running Tests

```bash
bundle exec rspec
```

## Code Style

We use [RuboCop](https://rubocop.org/) for linting:

```bash
bundle exec rubocop
```

## Pull Requests

1. Fork the repo and create your branch from `main`
2. Add tests for any new functionality
3. Ensure all tests pass
4. Run the linter
5. Submit your PR

## Reporting Issues

Use [GitHub Issues](https://github.com/MasonBachmann7/bugstack-ruby/issues).
